# 🖥️ Simple Java GUI

Este projeto foi desenvolvido como atividade final do curso introdutório de Java da Codedex.

A proposta era criar uma interface gráfica simples utilizando Java e aplicar os conceitos aprendidos ao longo do curso.

---

## 🚀 O que a aplicação faz

* Abre uma janela gráfica
* Permite digitar um nome
* Ao clicar no botão, exibe uma mensagem personalizada na tela

Exemplo:

```
Hello, Kaique!
```

---

## 🧠 O que eu pratiquei aqui

* Estrutura básica de um programa em Java
* Uso de `JFrame`, `JTextField`, `JButton` e `JLabel`
* Manipulação de eventos com `addActionListener`
* Organização do código separando em métodos
* Conceitos iniciais de Programação Orientada a Objetos

---

## 🎯 Objetivo

Consolidar os fundamentos de Java e entender como funciona a criação de interfaces gráficas e interação com o usuário.

Este projeto representa um dos primeiros passos da minha jornada como desenvolvedor.
